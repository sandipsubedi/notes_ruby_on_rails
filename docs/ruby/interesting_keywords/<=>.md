---
title: <=> SpaceShip Operator
---

### Used in comparison

```rb
# Origional
a = 1
b = 2

p -1 if a < b 
p 0 if a == b
p 1 if a > b

# Output:
-1
```
```rb
#  With spaceship operator
a = 1
b = 2
p a <=> b

# Output:
-1
```
