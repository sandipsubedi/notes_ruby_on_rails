# Ruby Version Manager:

### Change the version that you want to use:
If you want to use `2.6.3`

### Install:
```ruby
rvm install "ruby-2.6.3"
```

### Use:
```ruby
rvm use 2.6.3
```

### To make a ruby version default:
```ruby
rvm use --default 2.6.3
```

### Check which ruby is being used:
```ruby
rvm which ruby
```
